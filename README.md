# Bolt & Nut

Параметрическая модель болта и гайки. Полный рабочий процесс: моделирование в Fusion 360 → экспорт → рендер и анимация сборки.

![render](421a182a-80c6-4b98-8ead-f1bee691c470.PNG)

## Что сделано
- Модель болта и гайки, проверенные размеры и посадки.  
- Рендер сцены и GIF анимация сборки/закручивания.  
- Экспорт.
